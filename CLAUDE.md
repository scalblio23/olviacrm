# Working agreements for Claude

## Pull requests
- **Never merge a PR without explicit confirmation.** Before merging, post a
  summary of what's being merged (scope, files, risk), then wait. Only run the
  merge after the user explicitly says "merge it" (or equivalent). This applies
  to every PR, no exceptions.
- "Open a PR" / "ship it" means create or update the PR and stop — it is NOT
  merge authorization.
